TASK 5: Grid Gaps

Q1. Where does gap add space?
→ Gap adds space between grid items.
Q2. How is gap different from margin?
→ Gap adds space between items, margin adds space outside items.
Q3. Does gap affect container size?
→ No, it only affects spacing between items.
Q4. Can we use gap only for rows or columns?
→ Yes, using row-gap or column-gap.
Q5. Why is gap preferred in Grid layouts?
→ It keeps spacing clean and controlled.
