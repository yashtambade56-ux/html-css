TASK 4: Grid Rows

Task 4 Questions...
Q1. What does auto do in grid rows?
→ auto adjusts height based on content.

Q2. Which row grows based on content?
→ The row with auto value grows.

Q3. Can rows have different heights?
→ Yes, each row can have different height.

Q4. What happens if content is very large?
→ auto row increases height to fit content.

Q5. Where are grid rows useful in real layouts?
→ In headers, content sections, and footers.
