Q1. What problem does auto-fit solve?
→ It automatically adjusts the number of columns based on screen size.

Q2. Why is minmax important?
→ It sets minimum and maximum width for grid items.

Q3. Do items shrink too much?
→ No, minmax prevents items from becoming too small.

Q4. Is this layout mobile friendly?
→ Yes, it adapts automatically to small screens.

Q5. Why are media queries not needed here?
→ Grid layout adjusts automatically using auto-fit and minmax.
