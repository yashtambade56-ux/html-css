Q1. Why is 12-column grid popular?
→ Because it is easy to divide into halves, thirds, and quarters.

Q2. How does spanning help layout?
→ It allows elements to take more or less space as needed.

Q3. Where is this system used?
→ Used in websites, dashboards, and frameworks like Bootstrap.

Q4. Can we create layouts without this system?
→ Yes, but layout may be less consistent.

Q5. What happens if column count changes?
→ Layout structure changes and spans must be adjusted.
